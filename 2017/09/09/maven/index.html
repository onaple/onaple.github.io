常用难记命令

  1. mvn clean install -D maven.test.skip=true
  2. mvn clean package -U -DfailIfNoTests=false -f pom.xml
  1. 创建Maven的普通java项目：
    mvn archetype:create -DgroupId=packageName -DartifactId=projectName

  2. 创建Maven的Web项目：
    mvn archetype:create -DgroupId=packageName -DartifactId=webappName -DarchetypeArtifactId=maven-archetype-webapp

  3. 生成eclipse项目：mvn eclipse:eclipse

  4. 生成idea项目：mvn idea:idea
  5. 跳过测试: mvn install -Dmaven.test.skip=true
  6. 指定端口: mvn -Dmaven.tomcat.port=9090
